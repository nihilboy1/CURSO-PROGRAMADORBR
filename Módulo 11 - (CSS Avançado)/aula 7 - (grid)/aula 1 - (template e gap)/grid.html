display: grid;

define que eu estou usando grid layout

grid-template-columns:

recebe valores numericos, em %, px e fr. também pode receber auto.
essa propriedade define a dimensão das minhas colunas

grid-template-rows

recebe valores numericos, em %, px e fr. também pode receber auto.
essa propriedade define a dimensão das minhas linhas

grid-template:

esse short-hand do grid recebe primeiro as linhas e depois as colunas

gap:

define uma distancia entre as boxes, recebendo primeiro o valor para as rows e depois para as columns
